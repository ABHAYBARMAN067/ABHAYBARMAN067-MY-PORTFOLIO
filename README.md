# MY-PORTFOLIO-test

A  personal portfolio website built with Next.js, React, TypeScript, and Tailwind CSS.

## Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* ESLint

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Available Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
